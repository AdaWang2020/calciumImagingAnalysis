# External software packages

The folder `_external_programs` is for external software packages used by `calciumImagingAnalysis`.

The following are likely to be found here:
- GUI and display
	- `Fiji`
- Cell extraction
	- `CELLMax`
	- `CNMF`
	- `CNMF-E`
	- `CVX`
	- `EXTRACT`
- Motion correction
	- `NoRMCorre`
